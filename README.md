# Loan Approval Prediction

## Project Overview

This project aims to predict whether a loan application will be approved based on applicant information using machine learning techniques. The project covers the complete data science workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model building, evaluation, and business interpretation.

## Objective

To develop a predictive model that assists financial institutions in making efficient and data-driven loan approval decisions.

## Dataset

Loan Approval Prediction Dataset

Features:

* Gender
* Married
* Dependents
* Education
* Self_Employed
* ApplicantIncome
* CoapplicantIncome
* LoanAmount
* Loan_Amount_Term
* Credit_History
* Property_Area

Target Variable:

* Loan_Status

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Missing Value Treatment
4. Exploratory Data Analysis (EDA)
5. Feature Engineering
6. Label Encoding
7. Feature Scaling
8. Class Imbalance Handling using SMOTE
9. Model Building
10. Model Evaluation
11. Business Interpretation

## Machine Learning Models

* Logistic Regression
* Random Forest Classifier

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* ROC-AUC Score

## Key Findings

* Credit History is the strongest predictor of loan approval.
* Applicant Income positively influences approval chances.
* Property Area contributes to loan approval patterns.
* Random Forest achieved the best overall performance.
* Machine learning can effectively support loan approval decision-making.
